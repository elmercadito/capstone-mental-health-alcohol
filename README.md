# Mental Health and Alcohol Consumption (CCHS 2019–2020)

## Overview
Honours econometrics capstone analyzing the relationship between alcohol consumption patterns and self-reported mental health outcomes using Canadian survey microdata (CCHS 2019–2020).

This repo is structured to demonstrate reproducible applied research (cleaning → modelling → outputs) for employers.

## Data Access Note
Due to Statistics Canada microdata restrictions, raw CCHS files are not included in this repository.
All analysis code is provided and will run for users with authorized access to the dataset.

## Methodology (high-level)
- Descriptive statistics + subgroup comparisons  
- Multivariate regression models  
- Survey weights + bootstrap variance estimation (where applicable)  
- Robustness/sensitivity checks

## Repository Structure
- `code/` — Stata do-files (cleaning, analysis, models)
- `docs/` — variable dictionary + research notes
- `output/` — tables and figures generated by the analysis
